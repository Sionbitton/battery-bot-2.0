# Battery Bot

Battery Bot is a 3D browser driving game built with JavaScript and Three.js.

## Project Summary

The game started as a battery-based driving game and is being developed into an arcade-style driving game with an alien mood system, pickups, drop-offs, upgrades, recharge pads, and speed-based gameplay.

## Current Features

- 3D driving gameplay
- Arcade-style car movement
- Alien/mood system
- Mood drains while driving
- Pickup and drop-off loop
- Money system
- Garage upgrade system
- Recharge pads
- Navigation arrow/objective system
- Billboard objects
- Music toggle
- Game-over penalty
- Cartoon-style alien ejection animation

## Tech Used

- HTML
- CSS
- JavaScript
- Three.js
